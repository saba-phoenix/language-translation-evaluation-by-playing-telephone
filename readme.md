# Evaluating Language Translation Models by Playing Telephone (EMNLP 2025)

This repository contains code, data preparation utilities, trained models and evaluation scripts for our long paper published in the EMNLP 2025 Main Conference: [_“Evaluating Language Translation Models by Playing Telephone.”_](https://arxiv.org/pdf/2509.19611)

## Trained Models

### Quality Estimation Models

1. [QE-MR](https://huggingface.co/era28/qe-model-rotation-fixed)
2. [QE-FT-MQM](https://huggingface.co/era28/qe-ft-mqm)
3. [QE-LR-HD](https://huggingface.co/era28/qe-language-rotation-hd)
4. [QE-LR-LD](https://huggingface.co/era28/qe-language-rotation-ld)
5. [QE-LR-LDD](https://huggingface.co/era28/qe-language-rotation-ldd)
6. [QE-LR-HDD](https://huggingface.co/era28/qe-language-rotation-hdd)

### Reference Based Models

7. [REG-MR-MR](https://huggingface.co/era28/regression-model-rotation-hmref)
8. [REG-MR-REL](https://huggingface.co/era28/regression-model-rotation-relativeref)

### Unified Metrics Models

9. [UM-MR-HM](https://huggingface.co/era28/unifiedmetrics-model-rotation-hmref)
10. [UM-MR-REL](https://huggingface.co/era28/unifiedmetrics-model-rotation-relativeref)
11. [UM-FT-MQM](https://huggingface.co/era28/unifiedmetrics-model-rotation-hmref-ftmqm)

### Ablation Models

12. [Ablation-IT](https://huggingface.co/era28/ablation-just-iteration)
13. [Ablation-UNMD](https://huggingface.co/era28/ablation-asitis)
